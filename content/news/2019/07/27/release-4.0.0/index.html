Release 4.0.0

27 July 2019

This is Rya’s fourth Apache release, version 4.0.0. It is available as a source code download:

  • Source download from Apache
  • Verify: pgp sha512
  • Release notes
  • Central repository for Maven and other dependency managers

Verify using the corresponding pgp signature (using the committer file in KEYS). If you cannot do that, the sha512 hash file may be used to check that the download has completed okay.

For fast downloads, current source distributions are hosted on mirror servers; older source distributions are in the archive. If a download from a mirror fails, retry, and the second download will likely succeed.

For security, hash and signature files are always hosted at Apache.
